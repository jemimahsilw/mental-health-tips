# Mental Health Tips application

## Overview
The rate of mental health issues world wide is escalating at an alarming rate. many cases of suicide, self hate, bipolar disorders, drug and substance abuse, depression and many more. 
Inorder to help people deal and prevent these mental health issues, this platform, mental health tips application is developed so that people can get solutions to deal with mental health disorder. people can also share their views.

## features
- categories of mental health tips
- comments section for people to share their views

## technologies
- languages: PHP, HTML, CSS, JavaScript
- Database: MySQL

## Set up instructions
requirements: it runs on local server. In this case used xamp
- clone the repository
- run xamp and start apache and sql
- create databse "health_tips" either using cmd for MySQL or phpmyadmin
- in browser, run "localhost/mental-health-tips/"

## Future improvements
- addition of another user group, mental health specialist, to be updating the mental health tips
- addition of a feature to allow user upload their testimony videos on mental health and how they overcame the issue.
